# Web Technology Experiments

This repository contains a collection of practical projects and experiments completed for the Semester 4 Web Technology course. The experiments cover the fundamentals of HTML, CSS, JavaScript, PHP, and MySQL.

## Overview of Experiments

| Experiment | Project | Description | Links |
| :--- | :--- | :--- | :--- |
| **Experiment 1** | Portfolio Website | A basic portfolio website built with HTML only. | [View Project](./experiment1/experiment1.html) |
| **Experiment 2** | E-Commerce Website | A simple e-commerce site with product pages and cart, built in HTML. | [View Project](./experiment2/experiment2.html) |
| **Experiment 3** | Portfolio with CSS | Portfolio website enhanced with modern CSS styling and responsive layout. | [View Project](./experiment3/experiment3.html) |
| **Experiment 4** | E-Commerce with CSS | E-commerce site styled with CSS — colors, layout, and mobile responsiveness. | [View Project](./experiment4/experiment4.html) |
| **Experiment 5** | JavaScript Calculators | Basic computational and smartphone-style calculators built with JS. | [View Project](./experiment5/experiment5.html) |
| **Experiment 6** | Contact Form | A styled contact form with JS-powered submission display and copy functionality. | [View Project](./experiment6/experiment6.html) |
| **Experiment 7** | Event Handling | Event handling mechanisms — search, filter, capitalize, lowercase, and topper highlight on a student table. | [View Project](./experiment7/experiment7.html) |
| **Experiment 8** | PHP Form Handling | Form handling and validation for Exercise 1 and Exercise 2 using PHP. | [View Project](https://wtlab.gt.tc/experiment8/index.php) |
| **Experiment 9** | PHP + MySQL Product Management | Create, store, retrieve, and display product details using PHP and MySQL. | [View Project](https://wtlab.gt.tc/experiment9/index.php) |
| **Experiment 10** | Sessions and Cookies | Use PHP sessions and cookies to store and share user information across multiple pages. | [View Project](https://wtlab.gt.tc/experiment10/index.php) |

## Quick Links
- Experiments 1 through 7 are frontend projects viewable directly in the browser via their respective HTML files.
- Experiments 8 through 10 involve PHP and a database backend, and are currently hosted externally for demonstration.
